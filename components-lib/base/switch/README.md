# Switch 开关

开关切换组件。

## 使用方式

```html
<label class="switch">
  <input type="checkbox" class="switch-input">
  <span class="switch-slider"></span>
  <span class="switch-label">标签</span>
</label>
```

## 尺寸

| Class | 说明 |
|-------|------|
| `.switch-sm` | 小尺寸 |
| （默认） | 中尺寸 |
| `.switch-lg` | 大尺寸 |

## CSS 变量
