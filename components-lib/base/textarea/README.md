# Textarea 文本域

多行文本输入组件。

## 使用方式

```html
<textarea class="textarea" placeholder="请输入"></textarea>
```

## 带字数统计

```html
<div class="textarea-wrapper">
  <textarea class="textarea" maxlength="200"></textarea>
  <span class="textarea-count"><span>0</span>/200</span>
</div>
```

```css
--textarea-min-height: ;
```
