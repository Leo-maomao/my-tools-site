# Container 容器

页面内容容器，限制最大宽度并居中。

## 使用方式

```html
<div class="container">内容</div>
```

## 尺寸

| Class | 最大宽度 |
|-------|---------|
| `.container-sm` | 720px |
| `.container` | 1200px |
| `.container-lg` | 1400px |
| `.container-fluid` | 100% |

## CSS 变量

```css
--container-max-width: ;
--container-padding-x: ;
```
