# Card 卡片

卡片容器组件，用于展示内容块。

## 使用方式

```html
<div class="card">
  <div class="card-body">
    <h3 class="card-title">标题</h3>
    <p class="card-text">内容</p>
  </div>
</div>
```

## 变体

| Class | 说明 |
|-------|------|
| `.card-hoverable` | 可点击，有悬停效果 |
| `.card-selected` | 选中状态 |
| `.card-borderless` | 无边框 |
| `.card-compact` | 紧凑模式 |
| `.card-horizontal` | 水平布局 |
