# Tag 标签

用于标记和分类的标签组件。

## 使用方式

```html
<span class="tag">标签</span>
<span class="tag tag-primary">主要</span>
<span class="tag tag-success">成功</span>
```

## 颜色

| Class | 说明 |
|-------|------|
| （默认） | 灰色 |
| `.tag-primary` | 主色 |
| `.tag-success` | 成功 |
| `.tag-warning` | 警告 |
| `.tag-danger` | 危险 |
| `.tag-info` | 信息 |

## 样式

| Class | 说明 |
|-------|------|
| `.tag-outline` | 轮廓样式 |
| `.tag-rounded` | 圆角样式 |

## 尺寸

| Class | 说明 |
|-------|------|
| `.tag-sm` | 小 |
| （默认） | 中 |
| `.tag-lg` | 大 |

## CSS 变量

```css
--tag-padding-x: ;
--tag-padding-y: ;
--tag-font-size: ;
--tag-color: ;
--tag-bg: ;
--tag-border: ;
--tag-radius: ;
```
