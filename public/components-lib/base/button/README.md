# Button 按钮

通用按钮组件，支持多种变体和状态。

## 使用方式

```html
<button class="btn btn-primary">按钮文字</button>
```

## 变体

| Class | 说明 |
|-------|------|
| `.btn` | 基础按钮 |
| `.btn-primary` | 主要按钮 |
| `.btn-secondary` | 次要按钮 |
| `.btn-outline` | 轮廓按钮 |
| `.btn-text` | 文字按钮 |
| `.btn-danger` | 危险按钮 |
| `.btn-success` | 成功按钮 |

## 尺寸

| Class | 说明 |
|-------|------|
| `.btn-sm` | 小按钮 |
| （默认） | 中按钮 |
| `.btn-lg` | 大按钮 |
| `.btn-block` | 块级按钮 |

## 状态

| 状态 | 实现方式 |
|------|---------|
| 禁用 | `disabled` 属性 |
| 加载 | `.btn-loading` + spinner |
