# Progress 进度条

展示操作进度。

## CSS 变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `--progress-height` | `8px` | 高度 |
| `--progress-radius` | `9999px` | 圆角 |

## 修饰类

- `.progress-sm` - 小尺寸
- `.progress-lg` - 大尺寸
- `.progress-success/warning/error` - 颜色
- `.progress-striped` - 条纹效果
- `.progress-animated` - 动画效果
